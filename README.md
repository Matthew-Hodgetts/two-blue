#Two Blue Wordpress Theme
===



Getting Started
---------------

Clone or download

Run `yarn install`

Make sure you have a local build of wordpress set up and link your virtualhost in 'gulpfile.babel.js'

##Development

`yarn run dev` to commence with development

##Build

`yarn run prod` to initialise the build

Will make a dist folder and a .zip of the theme.
